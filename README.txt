Instructions for Use

(1) Type the following on the MATLAB command line: INLgui;

(2) Press the 'Select Folder' button and then use the prompt to select the folder that 
contains the *.xlsx files you are interested in plotting.

(3) Select the file(s) that you are interested in plotting. Hold the ctrl button to
select multiple files. Press the Load File(s) button.

(4) Select the variables you would like to plot, holding the CTRL button allows for 
multiple selections to be made. You must use the left-hand axis.

(5) Press the 'Plot Data' button. Checking the Overlay box will remove the time 
stamps from the data and plot them all from zero.
